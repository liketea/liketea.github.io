Scala 2.8 зҡ„йӣҶеҗҲжЎҶжһ¶жңүд»ҘдёӢзү№зӮ№пјҡ

  1. жҳ“з”ЁпјҡдҪҝз”Ё 20~50 дёӘж–№жі•зҡ„иҜҚжұҮйҮҸе°ұи¶ід»Ҙи§ЈеҶіеӨ§йғЁеҲҶзҡ„йӣҶеҗҲй—®йўҳпјӣ
  2. з®ҖжҙҒпјҡеҸҜд»ҘйҖҡиҝҮеҚ•зӢ¬зҡ„дёҖдёӘиҜҚжқҘжү§иЎҢдёҖдёӘжҲ–еӨҡдёӘеҫӘзҺҜпјӣ
  3. е®үе…ЁпјҡScala йӣҶеҗҲзҡ„йқҷжҖҒзұ»еһӢе’ҢеҮҪж•°жҖ§иҙЁж„Ҹе‘ізқҖеңЁзј–иҜ‘ж—¶е°ұеҸҜд»ҘжҚ•иҺ·з»қеӨ§еӨҡж•°й”ҷиҜҜпјӣ
  4. еҝ«йҖҹпјҡйӣҶеҗҲж“ҚдҪңе·Із»ҸеңЁзұ»еә“дёӯдјҳеҢ–иҝҮпјӣ
  5. йҖҡз”ЁпјҡйӣҶеҗҲзұ»жҸҗдҫӣдәҶеңЁдёҖдәӣзұ»еһӢдёҠзҡ„зӣёеҗҢж“ҚдҪңпјӣ

SeqгҖҒMapгҖҒSet жҳҜ Scala жңҖйҮҚиҰҒзҡ„дёүз§ҚйӣҶеҗҲзұ»пјҲе®№еҷЁпјүпјҢжӯӨеӨ–иҝҳжңү TupleгҖҒOption зӯүпјҢиҝҷдәӣдјҡеңЁеҗҺйқўе°ҸиҠӮйҖҗдёҖи®Іи§ЈпјҢжң¬иҠӮе°ҶжҢүз…§иҮӘйЎ¶еҗ‘дёӢзҡ„еұӮзә§з»“жһ„жқҘеӯҰд№ дёҚеҗҢйӣҶеҗҲзұ»зҡ„йҖҡз”Ёзү№жҖ§гҖӮ

еҸҜеҸҳ/дёҚеҸҜеҸҳзұ»еһӢпјҲMutable/Immutableпјү

Scala йӣҶеҗҲжЎҶжһ¶зі»з»ҹең°еҢәеҲҶдәҶеҸҜеҸҳзҡ„(mutable)е’ҢдёҚеҸҜеҸҳзҡ„(immutable)йӣҶеҗҲпјҢе№¶дё”еҸҜд»ҘеҫҲж–№дҫҝең°еңЁдёӨиҖ…д№Ӣй—ҙиҝӣиЎҢиҪ¬жҚўгҖӮдҪ еҸҜд»ҘеҜ№еҸҜеҸҳйӣҶеҗҲдёӯзҡ„е…ғзҙ иҝӣиЎҢеўһгҖҒеҲ гҖҒж”№ж“ҚдҪңпјҢдҪ д№ҹеҸҜд»ҘеҜ№дёҚеҸҜеҸҳзұ»еһӢжЁЎжӢҹиҝҷдәӣж“ҚдҪңпјҢдҪҶжҜҸдёӘж“ҚдҪңйғҪдјҡиҝ”еӣһдёҖдёӘж–°зҡ„йӣҶеҗҲпјҢеҺҹжқҘзҡ„йӣҶеҗҲдёҚдјҡеҸ‘з”ҹж”№еҸҳгҖӮ

йӣҶеҗҲзұ»зҡ„з»§жүҝж ‘

Scala жүҖжңүйӣҶеҗҲзұ»йғҪеҸҜд»ҘеңЁд»ҘдёӢеҢ…дёӯжүҫеҲ°пјҡ

  • scala.collectionпјҡеҢ…дёӯзҡ„йӣҶеҗҲж—ўеҸҜд»ҘжҳҜеҸҜеҸҳзҡ„д№ҹеҸҜд»ҘжҳҜдёҚеҸҜеҸҳзҡ„пјҢдёӢеӣҫеұ•зӨәдәҶиҝҷдёӘеҢ…дёӯжүҖжңүзҡ„йӣҶеҗҲзұ»пјҢиҝҷдәӣйғҪжҳҜй«ҳзә§жҠҪиұЎзұ»жҲ–зү№иҙЁпјҢе®ғд»¬йҖҡеёёжңүеҸҜеҸҳе’ҢдёҚеҸҜеҸҳдёӨз§Қе®һзҺ°ж–№ејҸ

  • scala.collection.immutable пјҡеҢ…дёӯзҡ„йӣҶеҗҲзұ»жҳҜдёҚеҸҜеҸҳзҡ„пјҢScalaдјҡй»ҳи®ӨеҜје…ҘиҝҷдёӘеҢ…пјҢиҝҷж„Ҹе‘ізқҖScalaй»ҳи®ӨдҪҝз”ЁдёҚеҸҜеҸҳйӣҶеҗҲзұ»пјҢеҪ“дҪ еҶҷдёӢ Set иҖҢжІЎжңүеҠ д»»дҪ•еүҚзјҖпјҢдҪ дјҡеҫ—еҲ°дёҖдёӘдёҚеҸҜеҸҳзҡ„ SetпјҢдёӢеӣҫеұ•зӨәдәҶиҝҷдёӘеҢ…дёӯжүҖжңүзҡ„йӣҶеҗҲзұ»

  • scala.collection.mutableпјҡеҢ…дёӯзҡ„йӣҶеҗҲзұ»жҳҜеҸҜеҸҳзҡ„пјҢеҰӮжһңдҪ жғіиҰҒдҪҝз”ЁеҸҜеҸҳзҡ„йӣҶеҗҲзұ»пјҢйҖҡз”Ёзҡ„еҒҡжі•жҳҜеҜје…Ҙscala.collection.mutableеҢ…еҚіеҸҜпјҢеҪ“дҪ дҪҝз”ЁжІЎжңүеүҚзјҖзҡ„ Set ж—¶д»Қз„¶жҢҮзҡ„жҳҜдёҖдёӘдёҚеҸҜеҸҳйӣҶеҗҲпјҢеҪ“дҪ дҪҝз”Ё mutable.Setж—¶жҢҮзҡ„жҳҜеҸҜеҸҳзҡ„йӣҶеҗҲзұ»пјҢдёӢеӣҫеұ•зӨәдәҶиҝҷдёӘеҢ…дёӯжүҖжңүзҡ„йӣҶеҗҲзұ»

  • scala.collection.genericпјҡеҢ…еҗ«дәҶйӣҶеҗҲзҡ„жһ„е»әеқ—пјҢйӣҶеҗҲзұ»е»¶иҝҹдәҶcollection.generic зұ»дёӯзҡ„йғЁеҲҶж“ҚдҪңе®һзҺ°

йӣҶеҗҲзұ»зҡ„йҖҡз”Ёж–№жі•

Scala дёӯзҡ„йӣҶеҗҲзұ»жңүд»ҘдёӢйҖҡз”Ёж–№жі•пјҡ

  • йӣҶеҗҲеҲӣе»әпјҡжҜҸдёҖз§ҚйӣҶеҗҲйғҪеҸҜд»ҘйҖҡиҝҮеңЁйӣҶеҗҲзұ»еҗҚеҗҺзҙ§и·ҹе…ғзҙ зҡ„ж–№ејҸиҝӣиЎҢеҲӣе»ә
  • toStringпјҡжүҖжңүйӣҶеҗҲзұ»йғҪеҸҜд»Ҙз”ЁtoStringзҡ„ж–№ејҸиҝӣиЎҢжү“еҚ°пјӣ
  • иҝ”еӣһзұ»еһӢдёҖиҮҙеҺҹеҲҷпјҡжүҖжңүйӣҶеҗҲзұ»зҡ„mapж–№жі•йғҪдјҡиҝ”еӣһзӣёеҗҢзұ»еһӢзҡ„йӣҶеҗҲзұ»пјҢдҫӢеҰӮпјҢеңЁдёҖдёӘListдёҠи°ғз”ЁmapдјҡеҸҲз”ҹжҲҗдёҖдёӘListпјҢеңЁSetдёҠи°ғз”ЁдјҡеҶҚз”ҹжҲҗдёҖдёӘSetпјҢд»ҘжӯӨзұ»жҺЁпјӣ
  • еӨ§еӨҡж•°зұ»еңЁйӣҶеҗҲж ‘з§ҚйғҪеӯҳеңЁдёүз§ҚеҸҳдҪ“пјҡrootгҖҒmutableгҖҒimmutableпјӣ

еҸҜйҒҚеҺҶзү№иҙЁпјҲTrait Traversableпјү

еҸҜйҒҚеҺҶпјҲTraversableпјүжҳҜе®№еҷЁпјҲcollectionпјүзұ»зҡ„жңҖй«ҳзә§еҲ«зү№иҙЁпјҢе®ғе”ҜдёҖзҡ„жҠҪиұЎж“ҚдҪңжҳҜforeachгҖӮforeach жҳҜ Traversable жүҖжңүж“ҚдҪңзҡ„еҹәзЎҖпјҢз”ЁдәҺйҒҚеҺҶе®№еҷЁдёӯжүҖжңүе…ғзҙ пјҢе№¶еҜ№жҜҸдёӘе…ғзҙ иҝӣиЎҢжҢҮе®ҡзҡ„ж“ҚдҪңпјҡ

иҰҒе®һзҺ° Traversable зҡ„е®№еҷЁзұ»д»…йңҖиҰҒе®ҡд№үдёҺд№Ӣзӣёе…ізҡ„ж–№жі•пјҢе…¶д»–жүҖжңүж–№жі•йғҪеҸҜд»Ҙд»Һ Traversable дёӯз»§жүҝпјҢTraversable е®ҡд№үдәҶи®ёеӨҡж–№жі•пјҡ

  1. зӣёеҠ ж“ҚдҪң++пјҲadditionпјүиЎЁзӨәжҠҠдёӨдёӘtraversableеҜ№иұЎйҷ„еҠ еңЁдёҖиө·жҲ–иҖ…жҠҠдёҖдёӘиҝӯд»ЈеҷЁзҡ„жүҖжңүе…ғзҙ ж·»еҠ еҲ°traversableеҜ№иұЎзҡ„е°ҫйғЁ
  2. Mapж“ҚдҪңжңүmapпјҢflatMapе’ҢcollectпјҢе®ғд»¬еҸҜд»ҘйҖҡиҝҮеҜ№е®№еҷЁдёӯзҡ„е…ғзҙ иҝӣиЎҢжҹҗдәӣиҝҗз®—жқҘз”ҹжҲҗдёҖдёӘж–°зҡ„е®№еҷЁ
  3. иҪ¬жҚўеҷЁпјҲConversionпјүж“ҚдҪңеҢ…жӢ¬toArrayпјҢtoListпјҢtoIterableпјҢtoSeqпјҢtoIndexedSeqпјҢtoStreamпјҢtoSetпјҢе’ҢtoMapпјҢе®ғд»¬еҸҜд»ҘжҢүз…§жҹҗз§Қзү№е®ҡзҡ„ж–№жі•еҜ№дёҖдёӘTraversable е®№еҷЁиҝӣиЎҢиҪ¬жҚў
  4. жӢ·иҙқпјҲCopyingпјүж“ҚдҪңжңүcopyToBufferе’ҢcopyToArrayгҖӮд»Һеӯ—йқўж„ҸжҖқе°ұеҸҜд»ҘзҹҘйҒ“пјҢе®ғд»¬еҲҶеҲ«з”ЁдәҺжҠҠе®№еҷЁдёӯзҡ„е…ғзҙ е…ғзҙ жӢ·иҙқеҲ°дёҖдёӘзј“еҶІеҢәжҲ–иҖ…ж•°з»„йҮҢ
  5. Size infoж“ҚдҪңеҢ…жӢ¬жңүisEmptyпјҢnonEmptyпјҢsizeе’ҢhasDefiniteSize
  6. е…ғзҙ жЈҖзҙўпјҲElement Retrievalпјүж“ҚдҪңжңүheadпјҢlastпјҢheadOptionпјҢlastOptionе’ҢfindгҖӮиҝҷдәӣж“ҚдҪңеҸҜд»ҘжҹҘжүҫе®№еҷЁзҡ„з¬¬дёҖдёӘе…ғзҙ жҲ–иҖ…жңҖеҗҺдёҖдёӘе…ғзҙ пјҢжҲ–иҖ…з¬¬дёҖдёӘз¬ҰеҗҲжҹҗз§ҚжқЎд»¶зҡ„е…ғзҙ гҖӮжіЁж„ҸпјҢе°Ҫз®ЎеҰӮжӯӨпјҢдҪҶд№ҹдёҚжҳҜжүҖжңүзҡ„е®№еҷЁйғҪжҳҺзЎ®е®ҡд№үдәҶд»Җд№ҲжҳҜвҖңз¬¬дёҖдёӘвҖқжҲ–вҖқжңҖеҗҺдёҖдёӘвҖңгҖӮдҫӢеҰӮпјҢйҖҡиҝҮе“ҲеёҢеҖјеӮЁеӯҳе…ғзҙ зҡ„е“ҲеёҢйӣҶеҗҲпјҲhashSetпјүпјҢжҜҸж¬ЎиҝҗиЎҢе“ҲеёҢеҖјйғҪдјҡеҸ‘з”ҹж”№еҸҳгҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢзЁӢеәҸжҜҸж¬ЎиҝҗиЎҢйғҪеҸҜиғҪдјҡеҜјиҮҙе“ҲеёҢйӣҶеҗҲзҡ„вҖқз¬¬дёҖдёӘвҖңе…ғзҙ еҸ‘з”ҹеҸҳеҢ–гҖӮеҰӮжһңдёҖдёӘе®№еҷЁжҖ»жҳҜд»ҘзӣёеҗҢзҡ„и§„еҲҷжҺ’еҲ—е…ғзҙ пјҢйӮЈиҝҷдёӘе®№еҷЁжҳҜжңүеәҸзҡ„гҖӮеӨ§еӨҡж•°е®№еҷЁйғҪжҳҜжңүеәҸзҡ„пјҢдҪҶжңүдәӣдёҚжҳҜпјҲдҫӢеҰӮе“ҲеёҢйӣҶеҗҲпјүвҖ“ жҺ’еәҸдјҡйҖ жҲҗдёҖдәӣйўқеӨ–ж¶ҲиҖ—гҖӮжҺ’еәҸеҜ№дәҺйҮҚеӨҚжҖ§жөӢиҜ•е’Ңиҫ…еҠ©и°ғиҜ•жҳҜдёҚеҸҜжҲ–зјәзҡ„гҖӮиҝҷе°ұжҳҜдёәд»Җд№ҲScalaе®№еҷЁдёӯзҡ„жүҖжңүе®№еҷЁзұ»еһӢйғҪжҠҠжңүеәҸдҪңдёәеҸҜйҖүйЎ№гҖӮдҫӢеҰӮпјҢеёҰжңүеәҸжҖ§зҡ„HashSetе°ұжҳҜLinkedHashSet
  7. еӯҗе®№еҷЁжЈҖзҙўпјҲsub-collection Retrievalпјүж“ҚдҪңжңүtailпјҢinitпјҢsliceпјҢtakeпјҢdropпјҢtakeWhilteпјҢdropWhileпјҢfilterпјҢfilteNotе’ҢwithFilterгҖӮе®ғд»¬йғҪеҸҜд»ҘйҖҡиҝҮиҢғеӣҙзҙўеј•жҲ–дёҖдәӣи®әж–ӯзҡ„еҲӨж–ӯиҝ”еӣһжҹҗдәӣеӯҗе®№еҷЁ
  8. жӢҶеҲҶпјҲSubdivisionпјүж“ҚдҪңжңүsplitAtпјҢspanпјҢpartitionе’ҢgroupByпјҢе®ғд»¬з”ЁдәҺжҠҠдёҖдёӘе®№еҷЁпјҲcollectionпјүйҮҢзҡ„е…ғзҙ еҲҶеүІжҲҗеӨҡдёӘеӯҗе®№еҷЁ
  9. е…ғзҙ жөӢиҜ•пјҲElement testпјүеҢ…жӢ¬жңүexistsпјҢforallе’ҢcountпјҢе®ғд»¬еҸҜд»Ҙз”ЁдёҖдёӘз»ҷе®ҡи®әж–ӯжқҘеҜ№е®№еҷЁдёӯзҡ„е…ғзҙ иҝӣиЎҢеҲӨж–ӯ
  10. жҠҳеҸ пјҲFoldsпјүж“ҚдҪңжңүfoldLeftпјҢfoldRightпјҢ/:пјҢ:\пјҢreduceLeftе’ҢreduceRightпјҢз”ЁдәҺеҜ№иҝһз»ӯжҖ§е…ғзҙ зҡ„дәҢиҝӣеҲ¶ж“ҚдҪң
  11. зү№ж®ҠжҠҳеҸ пјҲSpecific foldsпјүеҢ…жӢ¬sum, product, min, maxгҖӮе®ғд»¬дё»иҰҒз”ЁдәҺзү№е®ҡзұ»еһӢзҡ„е®№еҷЁпјҲж•°еҖјжҲ–жҜ”иҫғпјү
  12. еӯ—з¬ҰдёІпјҲStringпјүж“ҚдҪңжңүmkStringпјҢaddStringе’ҢstringPrefixпјҢеҸҜд»Ҙе°ҶдёҖдёӘе®№еҷЁйҖҡиҝҮеҸҜйҖүзҡ„ж–№ејҸиҪ¬жҚўдёәеӯ—з¬ҰдёІ
  13. и§ҶеӣҫпјҲViewпјүж“ҚдҪңеҢ…еҗ«дёӨдёӘviewж–№жі•зҡ„йҮҚиҪҪдҪ“гҖӮдёҖдёӘviewеҜ№иұЎеҸҜд»ҘеҪ“дҪңжҳҜдёҖдёӘе®№еҷЁе®ўи§Ӯең°еұ•зӨә

TraversableеҜ№иұЎзҡ„ж“ҚдҪңпјҡ

Iterator жңүдёӨдёӘж–№жі•иҝ”еӣһиҝӯд»ЈеҷЁпјҡgroupedе’ҢslidingпјҢиҝҷдәӣиҝӯд»ЈеҷЁиҝ”еӣһзҡ„дёҚжҳҜеҚ•дёӘе…ғзҙ пјҢиҖҢжҳҜеҺҹе®№еҷЁе…ғзҙ зҡ„е…ЁйғЁеӯҗеәҸеҲ—пјҢgroupedж–№жі•иҝ”еӣһе…ғзҙ зҡ„еўһйҮҸеҲҶеқ—пјҢslidingж–№жі•з”ҹжҲҗдёҖдёӘж»‘еҠЁе…ғзҙ зҡ„зӘ—еҸЈпјҡ
